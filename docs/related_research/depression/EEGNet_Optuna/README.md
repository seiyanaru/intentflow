# EEGNet + Optuna for Depression Classification

## 基本情報
- **著者**: Ștefana Duță, Alina Elena Sultana
- **発表年**: 2025
- **ジャーナル**: Sensors (MDPI)
- **PDF**: [sensors-25-02083.pdf](./sensors-25-02083.pdf)

## 概要
複数データセットを統合し、Optunaによるハイパーパラメータ最適化を行ったEEGNetベースのうつ病分類。独自データセットで93.27%の精度を達成。

## 主要貢献
- 異なるソースからのEEGデータセット統合
- 軽量ニューラルネットワークでポータブルシステムへの統合可能
- 前処理・分類の標準化パイプライン

## 我々の研究との関連
- **データセット統合**: クロスサブジェクト設定と関連
- **Optuna最適化**: ハイパーパラメータチューニングの参考
