# プロジェクト概要

## 研究フォーカス
- ドメインシフトに頑健な、オフラインEEG運動想起分類。
- テスト時の不確実性下でも安全に適応する OTTA / TTT 手法。
- 安全性と遅延制約を守るオンライン Active BCI パイプライン。

## 主な作業領域
- `intentflow/offline/`: モデル学習、評価、OTTAロジック、アブレーション。
- `intentflow/online/`: ライブ推論、ストリーム入力、レコーダ、ブロードキャスタ。
- `docs/research_progress/`: 日付付き進捗ノートと実装状況。
- `docs/related_research/`: 文献整理と研究ポジショニング。

## 基本原則
- 再現性優先: ベースラインと比較対象は安定に保つ。
- オンラインコードは安全性優先: 不確実なら fail closed。
- 次の比較を説明しやすくする実験構造を優先する。

## 典型的な進め方
1. 関連ドキュメントとコードを読む。
2. 比較設計または比較監査を行う。
3. 必要最小限の config または code path だけを変える。
4. 対象を絞った検証を走らせる。
5. 差分と不確実性を含む結果メモまたは進捗ノートを書く。
