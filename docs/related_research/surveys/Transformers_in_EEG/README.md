# Transformers in EEG Analysis: A Review

## 基本情報
- **著者**: Elnaz Vafaei, Mohammad Hosseini
- **発表年**: 2025
- **ジャーナル**: Sensors (MDPI)
- **PDF**: [sensors-25-01293.pdf](./sensors-25-01293.pdf)

## 概要
EEG分析におけるTransformerアーキテクチャの包括的レビュー。Motor Imagery、Seizure Detection、Emotion Recognitionの3つの主要タスクを対象。

## カバーするアーキテクチャ
1. **Time Series Transformer**: 時系列データ向け
2. **Vision Transformer (ViT)**: 画像ベースアプローチ
3. **Graph Attention Transformer**: グラフ構造活用
4. **Hybrid Models**: CNN+Transformer等

## 主要トピック
- データ拡張戦略
- 転移学習アプローチ
- 各タスクでの最新結果比較

## 我々の研究との関連
- TCFormerを含むTransformerベースEEGモデルの文脈を理解するのに有用
- 転移学習・適応手法のサーベイとして参考
