# Mamba with KAN for EEG Classification

## 基本情報
- **著者**: Zhe Li
- **発表年**: 2025
- **ジャーナル**: Frontiers in Sensors
- **PDF**: [fsens-6-1548729.pdf](./fsens-6-1548729.pdf)

## 概要
**Mamba** (State Space Model) に **Kolmogorov-Arnold Network (KAN)** と **Channel-Spatial Attention** を組み合わせたEEG分類モデル。

## アーキテクチャ
- Split-based Pyramidal Convolution
- Mamba (SSM backbone)
- KAN-Channel-Spatial Attention

## 我々の研究との関連
- Mamba系アーキテクチャのEEG応用例
- EEGMambaと比較可能
