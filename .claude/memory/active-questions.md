# 現在の重要論点

## 研究の大目的
- Active BCI の精度を上げる。
- Active BCI を軽量にし、リアルタイム運用しやすくする。
- テストタイム適応を使って、被験者差・セッション差・環境変動に強くする。
- その適応が、精度改善だけでなく安全性と安定性も保てることを示す。

## 現在の主研究テーマ
- OTTA / TTT を、Active BCI に載せられる計算量と遅延で実現できるか。
- 適応による改善が本物か、それとも一部条件だけの見かけの gain かを見極める。
- offline で効いた手法が、online や live EEG でも破綻しないかを確かめる。
- 生理学的妥当性や解釈可能性を保ちながら、適応判断を賢くできるかを探る。

## よく詰まる点
- 目先の OTTA 実装に集中しすぎて、最終目的である精度向上・軽量化・実運用性から外れること。
- 一度に複数因子を変えて、何が効いたのか分からなくなること。
- config、seed、comparison target が欠けた provenance 不十分なログや結果整理。
- 平均値だけ見て、被験者ごとの差、失敗例、online リスクを見落とすこと。

## 常に確認すべき判断軸
- この変更は、精度・軽量性・適応安定性のどれに効くのか。
- その gain は、runtime cost や実装複雑性の増加に見合うか。
- offline evidence だけで十分か、それとも online safety evidence も必要か。
- 文献アイデアを深く統合する前に、小さな ablation で検証できるか。
