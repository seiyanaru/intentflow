# 論文・ノート用ルール

- contribution、method、assumptions、evidence、limitations を分けて書く。
- 著者の主張と、このリポジトリで再現・確認された根拠を区別する。
- 要約には、problem setting、core method、comparison target、empirical support、weak points を含める。
- future work を書くときは、すぐにこの repo でできる action と、より speculative な研究方向を分ける。
- 研究ノートは、後で実験計画時に素早く走査できる長さに保つ。
