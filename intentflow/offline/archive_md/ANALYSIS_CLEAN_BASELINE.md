## 📉 リスク分析: なぜ一部被験者が悪化したか

### 仮説1: 過学習（Overfitting）
**証拠**:
- 被験者4: 87.50% → 81.25% (-6.25%)
- 以前は437エポックで停止していた

**メカニズム**:
```
早期終了（437epoch） → 汎化性能最大 → Test 87.50%
完全訓練（1000epoch） → 過学習開始 → Test 81.25%
```

**対策**:
- Learning rate decay確認
- Dropout率の調整
- Weight decay強化

### 仮説2: Validation/Test分布の不一致
**証拠**:
- 修正前: Val = Test（完全一致）
- 修正後: Val ⊂ Session_T, Test = Session_E（異なるセッション）

**メカニズム**:
- Session_TとSession_Eでデータ分布が異なる
- Valで良い性能 ≠ Testで良い性能

**対策**:
- Session間の分布差を調査
- Domain Adaptationの考慮

### 仮説3: 偶然性（Random Variation）
**証拠**:
- 被験者3,7,8の悪化は<2%（微小）
- Seed=0のみの結果

**対策**:
- 5-seed実験で統計的検証
