## 全体戦略: 2段階の論文投稿計画

現在の成果を最大限に活かしつつ、次のステップへスムーズに繋げるため、以下の2段階構成で進める。

### **Phase 1: 今回の論文 (Current Paper / v1)**
- **タイトル案**: "Pmax-SAL Gated OTTA: Safe Test-Time Adaptation for Motor Imagery EEG"
- **コアメッセージ**: **「Gating（選別）による安全性」**
    - 既存手法（MI-IASW）は「重み付け」で全データを使おうとするが、ノイズの多いEEGでは危険（負の転移）。
    - 提案手法は「Pmax(確率) × SAL(幾何学)」でデータを厳選し、**「精度低下を絶対に防ぐ（Safety First）」** アプローチをとる。
- **Future Workへの記述**:
    - 「Gatingの欠点として、捨ててしまったデータを活用できない点がある。」
    - 「これを解決するため、今後は **Self-Supervised Learning (Masked Reconstruction)** の導入を検討している。」
    - → これにより、今回のアプローチの正当性を主張しつつ、次への伏線を張る。

### **Phase 2: 次回の論文 (Next Paper / v2)**
- **タイトル案**: "Self-Supervised Test-Time Training for Robust BCIs via Masked Signal Reconstruction"
- **コアメッセージ**: **「Gating不要の全データ適応」**
    - Phase 1の課題（データ廃棄）を解決する。
    - 擬似ラベル（予測）に頼らず、**脳波の復元タスク**で適応するため、どんなデータも安全に学習に使える。

## 直近のタスク（Phase 1 完遂に向けて）

### 1. アブレーション研究 (Ablation Study)
提案手法の各構成要素（Pmax, SAL）の必要性を証明する。

| 手法 | 構成 | 予想結果 | 目的 |
| :--- | :--- | :--- | :--- |
| **Baseline** | TTAなし | 低精度 | 基準値 |
| **Pmax Only** | Pmaxフィルタのみ | 低〜中 | SALの必要性証明 |
| **Entropy Only** | Entropyフィルタのみ | 中 | Pmaxの優位性証明 |
| **Proposed** | **Pmax + SAL** | **最高** | **相乗効果の証明** |

### 2. Discussionの強化（負の転移・パラメータ感度）
- **負の転移**: BCIC 2bの一部で精度が下がる現象について考察。「ノイズが強すぎてGatingをすり抜けた」等の仮説を立てる。
- **パラメータ感度**: `pmax_threshold` や `sal_threshold` を変化させた時の安定性を示す。

### 3. 計算コストの数値化
- MI-IASWやCoTTAとの推論時間比較（ms単位）を行い、"Efficient" であることを証明する。

## 将来の技術詳細（Phase 2: Self-Supervised TTT）

### アプローチ概要: Masked Signal Reconstruction (MSR)
ラベルを使わずに適応を行うための具体的な手法。

1.  **Reconstruction Headの追加**:
    - モデル出力に、元の脳波波形（または特徴量）を復元するデコーダを追加。
2.  **マスキング適応 (Masked Adaptation)**:
    - 入力脳波の一部をランダムにマスク（0埋め）する。
    - モデルに「マスクされた部分」を予測・復元させる。
    - **損失関数**: $L = L_{cls} + \lambda L_{rec}$
        - 自信があるデータ（High Pmax/SAL）: 分類ロス＋復元ロスで学習。
        - **自信がないデータ（Low Pmax/SAL）**: **復元ロスのみ**で学習。→ **ここが革新的。**
3.  **メリット**:
    - 捨てていたデータ（自信がないデータ）も「波形の勉強用」として有効活用できる。
