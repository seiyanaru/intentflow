# ゼミ発表用 可視化チェックリスト

結果が出たら、要約だけで終わらず、発表に使える図を準備する。

## まず作る図
- 被験者ごとの性能比較棒グラフ
- baseline との差分棒グラフ
- 平均値と分散を示す箱ひげ図または error bar 図
- 代表的な成功例 / 失敗例の抜粋

## 条件付きで作る図
- 時系列で適応が変わるなら、trial または window ごとの confidence / entropy / gate 推移
- attention や channel importance があるなら、脳領域寄与図や topomap
- 特徴空間の変化を見せたいなら、t-SNE / UMAP
- クラスごとの崩れ方を見せたいなら、混同行列

## 図を作るときの観点
- その図で何を主張するのかを1文で言えること
- 平均だけでなく subject-wise のばらつきが見えること
- gain だけでなく regression や failure case も隠さないこと
- 図のタイトルだけで意味が分かること

## 発表前に確認すること
- 図と本文で使っている config 名、比較対象、metrics が一致しているか
- 被験者番号やクラス名に誤りがないか
- 図の保存先が結果ディレクトリまたは再利用しやすい場所にまとまっているか
